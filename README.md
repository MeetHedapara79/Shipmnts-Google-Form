## Google Forms
This is a google form created with ReactJs.

#### Features
- Add/Delete Elements
- Duplicate ELements
- Set input required
- Drag elements to reorder
- Multiple element types
    - TextInput
    - TextArea
    - RadioInput
    - NumberInput
    - DateInput
    - TimeInput

#### Usage
```bash
git clone {GIT_URL}
cd {PROJECTNAME}
npm install
npm start
```
